## Installation

Clone the repository:

```bash
git clone https://github.com/chetannagoriya/portfolio_chetan.git
```

Navigate to the project folder:

```bash
cd portfolio_chetan
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
